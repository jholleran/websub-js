# WebSub Implementaion
An experimental implementation of the [WebSub](https://www.w3.org/TR/websub/) spec written in Javascript.

# Build

```
npm install
```
# Run locally

Running the Hub:
```
node hub.js
```

Running the Publisher:
```
node publisher.js
```

Running the subscriber:
```
node subscriber.js <resource URL>
```
